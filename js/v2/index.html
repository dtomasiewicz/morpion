Morpion Solitaire

The game is simple-- create lines of adjacent Xs, the number of which is determined by the variant. Lines can intersect and join at the ends, but cannot overlap. You can only draw one line for each X you play, so choose wisely. The game is over when you can no longer place any more Xs. In the D variants, the rules are the same except lines running in the same direction cannot join at their endpoints.
